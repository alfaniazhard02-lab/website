Desain Web Statis

Desain web statis merupakan salah satu bentuk pengembangan website yang memiliki tampilan tetap dan isi yang tidak berubah-ubah, kecuali dilakukan pembaruan secara manual oleh pengembang. Website statis biasanya dibuat menggunakan bahasa pemrograman dasar seperti HTML dan CSS, tanpa melibatkan bahasa pemrograman server seperti PHP atau database. Jenis website ini sangat cocok digunakan untuk menampilkan informasi sederhana, seperti profil perusahaan, portofolio, atau halaman promosi..

ELECTRONIK

Elektronik merupakan cabang ilmu yang mempelajari tentang pengendalian arus listrik melalui berbagai komponen aktif, seperti dioda, transistor, dan sirkuit terpadu (IC). Ilmu ini berfokus pada bagaimana aliran elektron dapat dimanfaatkan untuk menjalankan suatu fungsi tertentu, misalnya dalam proses komunikasi, perhitungan, pengendalian, maupun hiburan.
